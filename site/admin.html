Browser Console

Read-Only Admin Console

Review project status snapshots and inspect generated SentinelEdge JSON reports in the browser.

This first console cut is intentionally read-only. It does not connect directly to a running device, and it exposes no control actions. It consumes exported JSON artifacts from the runtime: a status snapshot from cargo run -- status-json and SIEM-style report files from cargo run -- report.
